# 25maths-cie0580-worksheets

CIE 0580 Mathematics auto-generated worksheets.

## Structure

```
by-topic/          ← Topic-based worksheets
by-difficulty/     ← Difficulty-graded worksheets
by-section/        ← Syllabus section worksheets
custom/            ← Custom/teacher-requested worksheets
templates/         ← Worksheet LaTeX templates
```

## Generation

Worksheets are generated from `25maths-cie0580-json` data source,
compiled using LaTeX templates, and output as PDF.
